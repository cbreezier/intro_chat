Variables

Computers would be pretty boring if all they could only print. That's why we have variables! A variable can be thought of as a box - it can store a value that changes.

Creating a variable is simple.

>>>name = "John"
>>>age = 3
>>>height = 70

>>>print name
John
>>>print age
3
>>>print height
70

The same notation is used to change a variable.

>>>name = "John"
>>>print name
John
>>>name = "Bill"
>>>print name
Bill

There are different types of variables. For example, the variable name is a string (a letter or a word). The variables age and height are both integers (numbers).

Be careful with inverted commas! If you use them, Python will print out exactly what is in between the inverted commas, rather than the variable.

>>>day = Thursday
>>>print "day"
day
>>>print day
Thursday

Concatenation of Strings

Concatenate is just a fancy way of saying "joining together". Observe the following examples.

>>> print "I like cats"
I like cats
>>> print "I like" + "cats"
I likecats
>>> print "I like" + " cats"
I like cats

It also works with variables!

>>> animal = "dogs"
>>> print "I like " + animal
I like dogs
>>> animal = "rabbits"
>>> print "I like " + animal
I like rabbits
>>>colour = “blue”
>>>print “I like” + colour + rabbits
I like bluerabbits
>>>print “I like” + colour + “ “ + rabbits
I like blue rabbits

Similar to how mathematical operators can only be used on integers, string concatenation can only be done on strings. Notice how string concatenation uses the plus sign? This can get confusing since you're using the same symbol to add numbers and join words.

      >>> y = 5
      >>> print "orange"+y
      Traceback (most recent call last):
      File "< pyshell#71>", line 1, in <module>
      print "orange"+y
      TypeError: cannot concatenate 'str' and 'int' objects

This is the same example from before. Does it make sense to add “orange” and 5? No, you can only add numbers together. But what Python is really trying to do here is to concatenate, or join “orange” and 5, as you can see from the error message.

So, does it make sense to join "orange" and 5? You might expect it to print “orange5”, but it gives you the error message instead. This is because it will only join strings together. Compare the following examples.

>>>x = 5
>>>print "apple" + x
TypeError: cannot concatenate 'str' and 'int' objects

>>>x = "5"
>>>print "apple" + x
apple5

While this solution works, it can present some problems. What if you needed to use mathematical operations on x?

>>>x = "10"
>>>print "x + 5 is " + x + 5
TypeError: cannot concatenate 'str' and 'int' objects
>>>print "x+5 is " + (x+5)
TypeError: cannot concatenate 'str' and 'int' objects

Addition will only work if x is an integer. So we need x to be an integer when we are using it for maths, and for it to be a string when we are trying to print it. This can be done by typecasting.

>>>x = 10
>>> print "x + 5 is " + str(x+5)
x + 5 is 15

str is short for string. str() simply turns everything inside the brackets into a string. So first Python calculates x + 5. Then it turns the result into a string. And finally it concatenates the two strings together and prints them.

Raw Input

This allows the program to take in input from the user.

>>>print "What is your name?"
>>> raw_input()
Cassius
'Cassius'

This becomes more useful when you use it with variables.

>>>print "What day is it today?"
>>>day = raw_input()
Friday
>>>print "Today is " + day
What day is it today?
Thursday
Today is Thursday

The user input is a string by default. This can cause some issues. The program below is meant to multiply the user’s number by five.

>>>print "Enter your number:"
>>> number = raw_input()
>>> print number*5
Enter your number:
15
1515151515

Oh no! It treated the variable number as a string, and interpreted "print number*5 " as "print number five times".

We can get around this problem by typecasting. Simply add int( ) around the raw_input, and it will convert it from a string to an integer.

\ 
>>>print "Enter your number:"
>>> number = int(raw_input())
>>> print number*5

Another solution is:

>>>print "Enter your number:"
>>> number = raw_input()
>>> print int(number)*5

If statements

If statements in code work similarly to if statements in normal English. For example, let’s say we want to write a difficult mathematical quiz program.

>>>What is 1 + 2?
>>>answer = raw_input()
>>> if answer == 3:
print "correct!"

If the variable “answer” is equal to 3, print “correct!”

Note that if statements require a double equals sign (==).

Whenever you have a = sign, it means you are assigning a value to a variable.

Whenever you have a == sign, it means you are seeing whether something is True or False.

Else statements

The program above will only print if the user answers 3. To get it to print in all cases, we can use an else statement.

>>>What is 1 + 2?
>>>answer = raw_input()
>>> if answer == 3:
   print "correct!"
else:
   print "incorrect!"

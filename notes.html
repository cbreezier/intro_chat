The very basics - Variables

Computers would be pretty boring if they did the same thing every single time. That's why we have variables - literally things that vary. A variable can be thought of as a box - it can store a value that can change or vary.

It's easiest to learn through observation and practice so let's jump into some examples. 

        name = 'John'
        age = 3
        height = 7.89
        if (x in range(10)):
            # do something
